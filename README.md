# TDDE23 - Funktionell och imperativ programmering, del 1

My lab solutions for the first part of TDDE23 (Functional and Imperative Programming) at Linköping University.

## Course Structure
The course runs during the first half of the autumn semester (weeks 36-44), with one lab per week:

- **Lab 1** - Getting started with Python
- **Lab 2** - Conditions and iteration
- **Lab 3** - Dictionaries and recursion
- **Lab 4** - More recursion
- **Lab 5** - Image processing (larger assignment)

## Repository Structure
```
├── lab1/          # Introduction to Python
├── lab2/          # Conditions and iteration
├── lab3/          # Dictionaries and recursion
├── lab4/          # More recursion
└── lab5/          # Image processing (with OpenCV)
```

## About the Course
TDDE23 is the first programming course for Computer Science students at LiU. It covers both functional and imperative programming paradigms, with a focus on Python. The course runs in parallel with TDDE24 (which continues after the exam period in week 44).

[Course page](https://www.ida.liu.se/~TDDE23/planering.shtml)
